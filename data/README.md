### Data setup

Dataset source: Kaggle, "Massive Yahoo Finance Dataset"

How to obtain the data:
1. Create a Kaggle account at https://kaggle.com
2. Go to the dataset page: https://www.kaggle.com/datasets/iveeaten3223times/massive-yahoo-finance-dataset
3. Click "Download" to get the ZIP file
4. Extract the contents into the project `data/` directory. The `data/` folder is git-ignored and should not be committed.
